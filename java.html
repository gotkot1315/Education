การเปรียบเทียบการประมวลผลของการรันโค้ด

หน้าแรก ภาษา c++ ภาษา c# ภาษา java ภาษา python 
        โปรแกรม Java ที่ให้มามีฟังก์ชันต่าง ๆ ดังนี้:
        1.addMatrices(int[][] matrixA, int[][] matrixB):
            •	รับเมทริกซ์สองตัวเป็นอินทีเจอร์ และทำการบวกกันตำแหน่งที่เหมือนกันของเมทริกซ์ทั้งสอง
            •	สร้างและคืนเมทริกซ์ใหม่ที่เกิดจากการบวกนี้
        2.transposeMatrix(int[][] matrix):
            •	รับเมทริกซ์เป็นอินทีเจอร์ และทำการคำนวณและคืนเมทริกซ์ทรานสโพสของเมทริกซ์นั้น
            •	การทรานสโพสของเมทริกซ์ได้จากการสลับแถวกับคอลัมน์
        3.determinant(double[][] matrix):
            •	คำนวณและคืนค่าดีเทอร์มิแนนท์ของเมทริกซ์ที่เป็นสี่เหลี่ยมจัตุรัส
            •	ใช้การเข้าถึงโดยทฤษฎีบทการขยายคณะกำลังในการคำนวณดีเทอร์มิแนนท์
        4.getSubmatrix(double[][] matrix, int row, int col):
            •	คืนเมทริกซ์ย่อยที่ได้จากการลบแถวและคอลัมน์ที่ระบุออกจากเมทริกซ์ที่ให้มา
            •	ใช้ในการคำนวณดีเทอร์มิแนนท์เพื่อให้ได้คอ-แฟคเตอร์
        5.inverseMatrix(int[][] matrix):
            •	คำนวณเมทริกซ์ผกผันของเมทริกซ์ที่เป็นสี่เหลี่ยมจัตุรัส
            •	ตรวจสอบว่าเมทริกซ์เป็นสี่เหลี่ยมหรือไม่
            •	แปลงเมทริกซ์จำนวนเต็มเป็นเมทริกซ์ทศนิยม
            •	ตรวจสอบว่าดีเทอร์มิแนนท์ไม่เป็นศูนย์
            •	คำนวณเมทริกซ์แอดจูเกทและนำมันมาใช้ในการคำนวณเมทริกซ์ผกผัน
            •	แปลงเมทริกซ์ผกผันจำนวนทศนิยมกลับเป็นเมทริกซ์จำนวนเต็ม
        6.printMatrix(int[][] matrix) และ printMatrix(double[][] matrix):
            •	พิมพ์ค่าของเมทริกซ์ 2 มิติลงในคอนโซล
        7.main(String[] args):
            •	เมธอด main ใช้เพื่อทดสอบฟังก์ชันต่าง ๆ ในโปรแกรม
            •	กำหนดค่าเริ่มต้นให้กับเมทริกซ์ 3 ตัว (matrixA, matrixB, และ matrixC)
            •	พิมพ์เมทริกซ์ต้นฉบับ
            •	ทำการดำเนินการต่าง ๆ บนเมทริกซ์ (การบวก, การทรานสโพส, และการผกผัน)
            •	พิมพ์ผลลัพธ์
            •	รวมถึงการจัดการข้อผิดพลาดกรณีที่เมทริกซ์ไม่ใช่สี่เหลี่ยมหรือไม่สามารถผกผันได้
            •	จำนวนเวลาที่ใช้ในการดำเนินการถูกวัดด้วย System.currentTimeMillis()
