การเปรียบเทียบการประมวลผลของการรันโค้ด

หน้าแรก ภาษา c++ ภาษา c# ภาษา java ภาษา python 
        โปรแกรม Python ที่ให้มานั้นมีลักษณะเป็นภาษาที่ใช้ไลบรารี NumPy สำหรับการดำเนินการกับเมทริกซ์ 
        โดยมีฟังก์ชันต่าง ๆ ดังนี้: 
        1. add_matrices(matrixA, matrixB): 
            • รับเมทริกซ์ matrixA และ matrixB เข้ามา 
            • ใช้ฟังก์ชัน np.add() ใน NumPy เพื่อทำการบวกเมทริกซ์
            • คืนเมทริกซ์ผลลัพธ์ 
        2. transpose_matrix(matrix): 
            • รับเมทริกซ์ matrix เข้ามา 
            • ใช้ฟังก์ชัน np.transpose() ใน NumPy เพื่อทำการคำนวณและคืนเมทริกซ์ทรานสโพส 
            • คือการสลับแถวกับคอลัมน์ของเมทริกซ์ 
        3. determinant(matrix): 
            • รับเมทริกซ์ matrix เข้ามา 
            • ใช้ฟังก์ชัน np.linalg.det() ใน NumPy เพื่อคำนวณดีเทอร์มิแนนท์ของเมทริกซ์ 
        4. get_submatrix(matrix, row, col): 
            • รับเมทริกซ์ matrix และตำแหน่ง row และ col เข้ามา 
            • ใช้ np.delete() ใน NumPy เพื่อลบแถวและคอลัมน์ที่กำหนดออกจากเมทริกซ์
            • คืนเมทริกซ์ย่อยที่ได้ 
        5. inverse_matrix(matrix): 
            • รับเมทริกซ์ matrix เข้ามา 
            • คำนวณดีเทอร์มิแนนท์ของเมทริกซ์และตรวจสอบว่ามันไม่เป็นศูนย์ 
            • คำนวณเมทริกซ์ผกผันโดยใช้คอ-แฟคเตอร์และนำมันมาคูณด้วยค่าที่เป็นผกผัน 
            • คืนเมทริกซ์ผกผัน 
        6. print_matrix(matrix):
            • รับเมทริกซ์ matrix และพิมพ์ค่าของเมทริกซ์นี้ออกทางคอนโซล 
        7. main(): 
            • กำหนดค่าให้กับเมทริกซ์ matrixA, matrixB, และ matrixC 
            • พิมพ์เมทริกซ์ต้นฉบับ 
            • ทำการดำเนินการต่าง ๆ บนเมทริกซ์ (การบวก, การทรานสโพส, และการผกผัน) 
            • รวมถึงการจัดการข้อผิดพลาดกรณีที่เมทริกซ์ไม่ใช่สี่เหลี่ยมหรือไม่สามารถผกผันได้
            • จัดเก็บเวลาเริ่มต้นและสิ้นสุดการทำงานเพื่อวัดระยะเวลาที่ใช้ 
            • พิมพ์ผลลัพธ์และระยะเวลาที่ใช้ในการทำงาน โดยทั้งหมดนี้ใช้ไลบรารี NumPy 
            เพื่อทำให้การดำเนินการกับเมทริกซ์มีประสิทธิภาพและสะดวกมากขึ้นใน Python.
