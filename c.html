การเปรียบเทียบการประมวลผลของการรันโค้ด

หน้าแรก ภาษา c++ ภาษา c# ภาษา java ภาษา python 
        โปรแกรม C++ ที่ให้มานี้มีลักษณะเป็นคลาส MatrixOperations ที่มีฟังก์ชันต่าง ๆ 
        สำหรับการดำเนินการกับเมทริกซ์ และฟังก์ชัน main ที่ใช้เพื่อทดสอบฟังก์ชันเหล่านั้น ๆ 
        และวัดเวลาที่ใช้ในการทำงาน:
        1.addMatrices(const std::vector>& matrixA, 
            const std::vector>& matrixB):
            •	รับเมทริกซ์ matrixA และ matrixB เป็นพารามิเตอร์แบบค่าอ้างอิง
            •	ทำการบวกเมทริกซ์ matrixA และ matrixB และคืนผลลัพธ์
        2.transposeMatrix(const std::vector>& matrix):
            •	รับเมทริกซ์ matrix แบบค่าอ้างอิง
            •	ทำการคำนวณและคืนเมทริกซ์ทรานสโพสของ matrix
        3.determinant(const std::vector>& matrix):
            •	รับเมทริกซ์ matrix แบบค่าอ้างอิง
            •	คำนวณและคืนค่าดีเทอร์มิแนนท์ของเมทริกซ์ที่มีประกอบด้วยตัวเลขทศนิยม
        4.getSubmatrix(const std::vector>& matrix, int row, int col):
            •	รับเมทริกซ์ matrix, และตำแหน่ง row และ col
            •	คืนเมทริกซ์ย่อยที่ได้จากการลบแถวและคอลัมน์ที่ระบุออกจาก matrix
        5.inverseMatrix(const std::vector>& matrix):
            •	รับเมทริกซ์ matrix แบบค่าอ้างอิง
            •	ทำการตรวจสอบว่าเมทริกซ์เป็นสี่เหลี่ยมหรือไม่
            •	แปลงเมทริกซ์จำนวนเต็มเป็นเมทริกซ์ทศนิยม
            •	ตรวจสอบว่าดีเทอร์มิแนนท์ไม่เป็นศูนย์
            •	คำนวณเมทริกซ์แอดจูเกทและนำมันมาใช้ในการคำนวณเมทริกซ์ผกผัน
            •	แปลงเมทริกซ์ผกผันจำนวนทศนิยมกลับเป็นเมทริกซ์จำนวนเต็ม
        6.printMatrix(const std::vector>& matrix) 
            และ printMatrix(const std::vector>& matrix):
            •	รับเมทริกซ์ matrix แบบค่าอ้างอิง
            •	พิมพ์ค่าของเมทริกซ์นี้ออกทางคอนโซล
        7.main():
            •	กำหนดค่าให้กับเมทริกซ์ matrixA, matrixB, และ matrixC
            •	พิมพ์เมทริกซ์ต้นฉบับ
            •	ทำการดำเนินการต่าง ๆ บนเมทริกซ์ (การบวก, การทรานสโพส, และการผกผัน)
            •	รวมถึงการจัดการข้อผิดพลาดกรณีที่เมทริกซ์ไม่ใช่สี่เหลี่ยมหรือไม่สามารถผกผันได้
            •	วัดและพิมพ์ระยะเวลาที่ใช้ในการทำงาน
