การเปรียบเทียบการประมวลผลของการรันโค้ด

หน้าแรก ภาษา c++ ภาษา c# ภาษา java ภาษา python 
        โปรแกรม C# ที่ให้มานี้มีคลาส MatrixOperations ที่มีฟังก์ชันต่าง ๆ สำหรับการดำเนินการกับเมทริกซ์ 
        และฟังก์ชัน Main ที่ใช้เพื่อทดสอบและวัดเวลาที่ใช้ในการทำงาน:
        1.Main():
            •	สร้างเมทริกซ์ matrixA, matrixB, และ matrixC ที่มีขนาด 3x3, 3x3, และ 2x2 ตามลำดับ
            •	สร้าง Stopwatch เพื่อวัดเวลาที่ใช้ในการทำงาน
            •	แสดงเมทริกซ์ต้นฉบับ matrixA, matrixB, และ matrixC
            •	ทำการบวกเมทริกซ์ matrixA และ matrixB และแสดงผลลัพธ์
            •	ทำการทรานสโพสเมทริกซ์ matrixA และแสดงผลลัพธ์
            •	ทำการผกผันเมทริกซ์ matrixC (หากเมทริกซ์ไม่สามารถผกผันได้ จะแสดงข้อผิดพลาด)
            •	หยุดและแสดงเวลาที่ใช้ทั้งหมด
        2.PrintMatrix(double[,] matrix):
            •	รับเมทริกซ์ matrix และแสดงค่าในเมทริกซ์นั้นทางคอนโซล
        3.AddMatrices(double[,] matrixA, double[,] matrixB):
            •	รับเมทริกซ์ matrixA และ matrixB
            •	บวกทุกตัวที่อยู่ในเมทริกซ์ matrixA และ matrixB และสร้างเมทริกซ์ผลลัพธ์
        4.TransposeMatrix(double[,] matrix):
            •	รับเมทริกซ์ matrix
            •	สร้างและคืนเมทริกซ์ทรานสโพสของ matrix
        5.InvertMatrix(double[,] matrix):
            •	รับเมทริกซ์ matrix
            •	ตรวจสอบว่าเมทริกซ์เป็นเมทริกซ์สี่เหลี่ยมหรือไม่
            •	สร้างเมทริกซ์เพิ่มเติม (augmented matrix) จาก matrix โดยเพิ่มเติมหนึ่งหลักของ
                ไอเดนติตีแต่ละแถว
            •	ใช้วิธีการจอร์แดนเอลิมิเนชัน (Gauss-Jordan elimination) เพื่อหาเมทริกซ์ผกผัน
            •	ดึงเมทริกซ์ผกผันจาก augmented matrix และคืนค่า
